## Transport: HTTP

This server listens on `http://localhost:3000/mcp` (override the port with the
`PORT` env var). Point an MCP client at that URL, or smoke-test it:

```bash
curl -s http://localhost:3000/mcp \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list"}'
```

```json
{
  "mcpServers": {
    "petstore-mcp": {
      "type": "http",
      "url": "http://localhost:3000/mcp"
    }
  }
}
```

**Claude Code** — open this folder in Claude Code; it auto-detects `.mcp.json`. Start the server first with `npm start`.

**Claude Desktop** — paste the block above into `claude_desktop_config.json`, then restart the app.

## Tools

- `list_pets` — List all pets
- `create_pet` — Create a pet
- `get_pet_by_id` — Fetch a pet by ID
- `delete_pet` — Delete a pet by ID

Each tool validates its parameters with [Zod](https://zod.dev), then runs a real
implementation — an HTTP call to `BASE_URL`.

## Security

This is a standard MCP server. For zero-trust auth (JWT guard) and a deception
canary, regenerate with `mcpfoundry ... --secure`, or place it behind the ZTAI
firewall.
